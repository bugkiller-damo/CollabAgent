## 一、新手引导与首次体验

### 1.1 欢迎页 / Onboarding Wizard

> 难度：中 | 工时：1 天 | 感知度：⭐⭐⭐⭐⭐

**现状**：注册后直接跳 `/channels/general`，用户不知道能做什么。

**建议**：首次登录展示 3 步引导：
1. "创建你的第一个频道" → 引导创建频道
2. "@agent 试试 AI 助手" → 演示 @agent 功能
3. "邀请团队成员" → 分享链接

### 1.2 注册流程优化

> 难度：低 | 工时：2 小时 | 感知度：⭐⭐⭐⭐

- 注册成功后弹 toast "注册成功，正在跳转" 而非静默跳转
- 密码强度实时指示条（弱/中/强）
- 用户名实时可用性检查（输入时调 API 检查是否重复）

### 1.3 默认内容填充

> 难度：低 | 工时：1 小时 | 感知度：⭐⭐⭐

- #general 频道预置欢迎消息（Markdown 格式，含使用提示）
- 侧栏首次显示 tooltip："点击频道开始聊天"

## 二、界面交互打磨

### 2.1 响应式适配

> 难度：中 | 工时：2 天 | 感知度：⭐⭐⭐⭐

**现状**：仅适配桌面端，移动端布局错乱。

**建议**：
- 移动端：侧栏折叠为汉堡菜单
- 平板端：侧栏缩窄为图标模式
- 消息输入框：移动端 fixed 底部 + 安全区适配

### 2.2 加载与空状态

> 难度：低 | 工时：半天 | 感知度：⭐⭐⭐

| 场景 | 现状 | 建议 |
|---|---|---|
| 频道列表加载中 | 空白 | Skeleton 骨架屏 |
| 消息加载中 | "暂无消息" | 加载动画 |
| 网络断开 | 黄条 "连接中断" | 优化样式 + 自动重试倒计时 |
| Agent 思考中 | 无反馈 | 输入框旁显示"Agent 正在输入..." |

### 2.3 消息体验增强

> 难度：中 | 工时：2 天 | 感知度：⭐⭐⭐⭐

- **Markdown 渲染** — 代码高亮、链接可点击、列表格式化
- **消息时间分组** — 超过 5 分钟的消息显示时间分隔线
- **未读消息标记** — 进入频道自动滚到第一条未读
- **@mention 高亮** — 别人 @ 你的消息用蓝色背景标记
- **消息编辑/删除** — hover 显示编辑和删除按钮（自己的消息）

### 2.4 快捷键

> 难度：低 | 工时：半天 | 感知度：⭐⭐⭐

| 快捷键 | 功能 |
|---|---|
| `Ctrl+K` | 快速切换频道 |
| `Ctrl+Enter` | 发送消息 |
| `Esc` | 关闭线程面板 |
| `Ctrl+Shift+A` | 快速 @agent |

### 2.5 主题切换

> 难度：低 | 工时：半天 | 感知度：⭐⭐⭐

- 深色/浅色模式切换
- 跟随系统主题

## 三、功能补全

- 拖拽上传 / 粘贴图片

- 文件类型图标 + 下载按钮
- 50MB 限制

### 3.4 通知系统

> 难度：中 | 工时：1 天 | 感知度：⭐⭐⭐

- 浏览器 Notification API（被 @ 时桌面通知）

- "有人 @ 了你" 高亮提示

### 3.5 消息搜索

> 难度：中 | 工时：1 天 | 感知度：⭐⭐⭐

- 频道内搜索（pg_trgm 全文索引已建）

- 点击结果跳转到对应消息

- `/api/health` 已有 → 前端"系统状态"页展示

## 五、稳定性与容错

### 5.1 错误边界

> 难度：低 | 工时：2 小时 | 感知度：⭐⭐⭐

**现状**：React 组件崩溃 → 白屏。

**建议**：每个路由加 ErrorBoundary，崩溃时显示"出错了，点此重试"。

### 5.2 API 请求重试

> 难度：低 | 工时：1 小时 | 感知度：⭐⭐

- 网络错误自动重试 3 次（指数退避）
- 失败 toast 提示

### 5.3 离线降级

> 难度：中 | 工时：半天 | 感知度：⭐⭐⭐

- WebSocket 断开 → HTTP 轮询 fallback
- 离线时显示"你已离线"横幅
- 恢复后自动补拉错过的消息

### 5.4 表单校验

> 难度：低 | 工时：2 小时 | 感知度：⭐⭐

- 所有输入框加前端校验（Zod schema）
- 提交前拦截 + 行内错误提示

## 六、安全加固

### 6.1 Rate Limiting

> 难度：低 | 工时：1 小时 | 感知度：⭐（对用户透明）

- 登录接口：5 次/分钟/IP
- 注册接口：3 次/分钟/IP
- 发消息：30 次/分钟/用户

### 6.2 XSS 防护

> 难度：低 | 工时：1 小时 | 感知度：⭐（对用户透明）

- 消息内容 HTML 转义
- Markdown 渲染限制危险标签

### 6.3 Session 管理

> 难度：低 | 工时：1 小时 | 感知度：⭐⭐

- "活动会话"页面，查看当前登录设备
- 远程踢下线
