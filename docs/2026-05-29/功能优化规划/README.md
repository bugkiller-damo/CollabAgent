## 优先级建议

### P0 — 本周必做（体验底线）
- @提及自动补全
- Shift+Enter 换行发送
- 创建频道弹窗
- 发送失败重试
- 全局面包屑/错误边界

### P1 — 两周内（显著提升）
- Markdown 渲染 + 代码高亮
- 图片/文件拖拽上传
- Loading 骨架屏
- 空状态引导
- 密码强度指示条

### P2 — 一月内（锦上添花）
- 深色/浅色主题
- Emoji 反应
- 频道成员管理
- 虚拟滚动
- 消息编辑
- httpOnly Cookie

### P3 — 有精力再做
- 浏览器通知
- 链接预览
- 响应式移动端
- PWA 离线支持
- 邮箱验证
