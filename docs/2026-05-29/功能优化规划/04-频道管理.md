# 频道管理

## 创建频道

### 现状
只能通过数据库或 API 手动创建频道。

### 优化项
| 功能 | 说明 | 优先级 |
|---|---|---|
| 创建频道弹窗 | 侧栏 "+" 按钮 → 弹窗输入名称/描述/可见性 | 高 |
| 名称自动格式化 | 输入中文 → 自动转拼音 + 小写 + 连字符 | 中 |
| 频道模板 | "项目频道"/"团队频道"预设 | 低 |

### 实现要点
- `<CreateChannelModal>` 组件
- POST /api/channels → 创建成功 → 刷新频道列表
- 公开/私有选择开关

## 频道设置

### 优化项
| 功能 | 说明 | 优先级 |
|---|---|---|
| 修改名称/描述 | 频道页顶部齿轮按钮 | 中 |
| 修改可见性 | 公开 ↔ 私有切换 | 中 |
| 归档频道 | 归档后不可发消息，但仍可查看 | 低 |
| 删除频道 | 管理员操作（二次确认） | 低 |

### 实现要点
- PATCH /api/channels/:id 界面
- 权限检查：仅管理员可修改

## 成员管理

### 优化项
| 功能 | 说明 | 优先级 |
|---|---|---|
| 成员列表 | 频道右侧面板显示成员 + Agent | 中 |
| 邀请成员 | 输入用户名 → 发送邀请 | 中 |
| 移除成员 | 管理员操作 | 低 |
| 角色分配 | 管理员/普通成员 | 低 |

### 实现要点
- `/api/channels/:id/members` 数据已可用
- 右侧面板组件 `<ChannelMembersPanel>`
- 邀请：POST `/api/channels/:id/invite`
