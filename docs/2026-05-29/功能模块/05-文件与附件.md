# 文件与附件

## 后端 (`packages/server/src/routes/attachments.ts`)

### 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/attachments/upload` | 上传文件（multipart） |
| GET | `/api/attachments/:id` | 下载文件 |

### 数据模型

```
attachments (
  id UUID PK,
  uploader_id UUID,
  uploader_type VARCHAR(10),
  filename VARCHAR(500),
  mime_type VARCHAR(100),
  size_bytes BIGINT,
  storage_key TEXT,           — 文件路径
  storage_url TEXT,           — 访问URL
  created_at TIMESTAMPTZ
)

message_attachments (
  message_id UUID → messages,
  attachment_id UUID → attachments,
  PRIMARY KEY (message_id, attachment_id)
)
```

### 约束

- 最大 50MB
- MIME 自动检测
- 支持格式: JPEG, PNG, GIF, WebP, PDF, TXT, MD, JSON, CSV
