# 用户系统与设置优化

## 现状问题

| 问题 | 影响 |
|------|------|
| 登录页无密码可见切换 | 输入密码无法确认 |
| 注册表单只有用户名和密码 | 缺少邮箱（服务端已支持） |
| 设置页无头像上传 | UI 已有但功能未通 |
| 无通知偏好设置 | 所有消息都推送，无开关 |
| 用户列表/在线状态缺失 | 不知道频道里有谁 |

## 优化方案

### 优先 P0

1. **登录页密码可见切换** — 密码框右侧 👁 按钮
2. **注册表单加邮箱+密码确认** — 前端校验 + 后端已有支持
3. **头像上传** — 设置页支持文件选择/拖拽上传（已有 `/api/attachments/upload`）

### P1

4. **频道成员列表** — 侧栏右侧或顶部显示在线用户头像
5. **用户在线状态** — WebSocket `agent:status` 事件驱动绿/灰圆点
6. **通知设置** — 免打扰模式 / 仅 @提及通知

### P2

7. **主题切换** — 暗色/亮色主题（TailwindCSS dark mode）
8. **语言切换** — 中/英文切换
9. **快捷键自定义** — 用户可修改快捷键
