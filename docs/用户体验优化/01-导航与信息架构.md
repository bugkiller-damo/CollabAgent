# 导航与信息架构优化

## 现状问题

| 问题 | 影响 |
|------|------|
| 频道切换时 URL 已变但无视觉反馈 | 用户不知道自己在哪个频道 |
| 线程展开后无"返回顶部" | 长线程滚动不便 |
| 侧栏 DM 区域始终显示"暂无私信" | 死区，无用信息 |
| 移动端无响应式布局 | 手机无法使用 |
| 无面包屑导航 | 深层页面无返回路径 |

## 优化方案

### 优先 P0（重要且简单）

1. **侧栏 DM 区改为"成员列表"** — 列出频道内活跃用户，点击可发起 DM
2. **频道标题加在线人数** — `#general · 3 online`
3. **移动端汉堡菜单** — 768px 以下隐藏侧栏，顶部加折叠按钮

### P1（中等工作量）

4. **面包屑导航** — 频道页顶部 `Home > #general > Thread`
5. **键盘快捷键** — `Ctrl+K` 快速搜索频道/用户
6. **未读消息跳转** — 点击频道名跳转到第一条未读

### P2（后期）

7. **频道分组/收藏** — 频道多时可折叠分组
8. **消息固定/置顶** — 管理员可 pin 消息到频道顶部
