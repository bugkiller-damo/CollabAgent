# 频道与消息体验优化

## 1. Markdown 渲染增强 (P0 · 3h)

**现状**: 消息内容以纯文本展示，Markdown 未渲染。

**优化项**:

| 项 | 说明 |
|---|---|
| 代码块高亮 | `react-markdown` + `rehype-highlight`（依赖已装） |
| 内联代码样式 | `` `code` `` 灰色背景区分 |
| 链接可点击 | 自动识别 URL 为可点击链接 |
| @提及高亮 | `@username` 蓝色高亮 + 可点击跳转 |
| #频道引用 | `#channel` 渲染为频道链接 |
| 表格渲染 | GFM 表格支持（`remark-gfm` 已装） |

## 2. @提及自动补全 (P0 · 3h)

**现状**: 输入 `@` 无任何提示。

**实现**: 
- 输入 `@` 触发弹窗，列出频道成员 + Agent
- 键盘 ↑↓ 选择，Enter 确认
- 从 `/api/channels/{name}/members` 获取成员列表

## 3. 消息操作增强 (P1 · 2h)

| 项 | 说明 |
|---|---|
| 右键菜单 | 消息上右键 → 复制/引用/删除（自己的消息） |
| 消息编辑 | 自己的消息可编辑（`PATCH /api/messages/{id}`） |
| 消息删除 | 软删除，显示"此消息已删除" |
| 长按复制 (移动端) | 触屏设备长按复制 |

## 4. 文件拖拽上传 (P1 · 3h)

**现状**: 仅文本输入。

**实现**:
- 输入框区域支持拖拽文件
- 粘贴图片自动上传（Clipboard API）
- 上传进度条
- 图片预览（小于 5MB 直接显示缩略图）

## 5. 未读消息提醒 (P1 · 2h)

| 项 | 说明 |
|---|---|
| 浏览器标题闪烁 | 新消息时标题变成 `[N] CollabAgent` |
| 桌面通知 | `Notification API`，需用户授权 |
| 频道红点 | 已有未读计数，加动画（跳动红点） |
| @我的高亮 | 被 @ 的消息黄色背景高亮 |

## 6. 消息搜索 (P2 · 4h)

**现状**: 服务端有 `/api/messages/search`，前端无 UI。

**前端新增**:
- 顶部搜索栏（`Ctrl+K` 唤起）
- 搜索结果列表（高亮关键词）
- 点击结果跳转到对应消息位置

## 7. 表情反应增强 (P2 · 1h)

| 项 | 说明 |
|---|---|
| 快捷反应 | hover 消息显示常用 👍👀✅❌ |
| 反应选择器 | 点击 `+` 展开 emoji picker |
| 反应计数 | 显示每种反应的人数 |
