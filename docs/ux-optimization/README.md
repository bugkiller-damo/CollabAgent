## 优先级概览

| 优先级 | 域 | 关键项 |
|---|---|---|
| P0 | 频道与消息 | 消息 Markdown 渲染完整化、@提及自动补全 |
| P0 | 登录注册 | 第三方 OAuth 登录（GitHub/Google） |
| P1 | 导航与布局 | 全局搜索、响应式移动端适配 |
| P1 | 稳定性 | 全局错误边界、API 请求重试 |
| P2 | 任务系统 | 看板拖拽交互、任务筛选/排序 |
| P2 | 设置与管理 | Admin 频道/成员管理页面 |
