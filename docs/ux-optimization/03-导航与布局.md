# 导航与布局优化

## 1. 全局搜索 (P1 · 3h)

**现状**: 无全局搜索入口。

**实现**:
- 顶部栏 `Ctrl+K` 唤起搜索弹窗
- 搜索类型: 频道、消息、用户、文件
- 输入即搜，结果分组展示
- ↑↓ 选择，Enter 跳转

## 2. 键盘快捷键 (P1 · 2h)

| 快捷键 | 功能 |
|---|---|
| `Ctrl+K` | 全局搜索 |
| `Ctrl+Enter` | 发送消息 |
| `Ctrl+Shift+L` | 切换频道列表 |
| `Ctrl+1~9` | 快速切换到第 N 个频道 |
| `Escape` | 关闭弹窗/清空输入 |
| `↑` (空输入框) | 编辑上一条消息 |

## 3. 响应式移动端适配 (P1 · 5h)

**现状**: 仅桌面端可用。

**移动端布局**:
- 侧栏默认隐藏，汉堡菜单唤起
- 消息列表全宽，输入框固定底部
- 手势: 右滑打开侧栏，左滑关闭
- 触摸反馈: 按钮最小 44×44px 触摸区域

**TailwindCSS 断点**:
```
sm (640px): 手机横屏 → 侧栏可展开
md (768px): 平板 → 侧栏 200px
lg (1024px): 桌面 → 侧栏 260px + 右侧面板
```

## 4. 侧栏优化 (P2 · 2h)

| 项 | 说明 |
|---|---|
| 频道排序 | 拖拽重新排序（本地存储） |
| 频道分组 | 支持创建频道文件夹 |
| 在线人数 | 每个频道显示在线成员数 |
| 收藏频道 | 星标频道置顶 |
| 最近消息预览 | 频道名下方灰色显示最后一条消息 |
| 侧栏折叠 | 折叠为仅图标模式（30px 宽） |
