# 设置与管理优化

## 1. 个人设置完善 (P1 · 3h)

**现状**: 仅个人资料页可用。

**新增页面**:

| 页面 | 功能 |
|---|---|
| `/settings/avatar` | 头像上传/裁剪 |
| `/settings/notifications` | 通知偏好（桌面/邮件/频道静音） |
| `/settings/appearance` | 主题切换（暗色/亮色/跟随系统） |
| `/settings/sessions` | 查看/管理登录设备 |

## 2. Admin 频道管理 (P2 · 3h)

**现状**: 仅入口卡片，无实际功能。

**实现**:
- 频道列表（名称、成员数、创建时间）
- 创建频道（名称、描述、公开/私有）
- 编辑/归档/删除频道
- 成员管理（添加/移除/角色分配）

## 3. Admin Agent 管理完善 (P1 · 2h)

**现状**: `AgentAdmin` 页面已有创建/列表/状态。

**优化**:
- Agent 详情页: 模型选择、系统 prompt 编辑、能力配置
- Agent 活动日志: 最近消息、工具调用次数、token 消耗
- Agent 启动/停止（已有开关按钮，待接后端）

## 4. Admin 成员管理 (P2 · 2h)

- 成员列表（角色、加入时间、在线状态）
- 邀请成员（生成邀请链接/邮箱邀请）
- 角色变更（Member ↔ Admin ↔ Owner）
- 移除成员

## 5. 服务器设置 (P2 · 1h)

- 服务器名称/描述修改
- 服务器图标
- 删除服务器确认流程
