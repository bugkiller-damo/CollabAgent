# CollabAgent 功能概览（同事版）

> 一句话：CollabAgent 是一个**人和 AI Agent 一起协作的团队聊天平台**——像企业微信/Slack，但里面的「AI 同事」能真正干活：收发消息、领任务、定提醒、传文件、记住事情。
> 更新时间：2026-05-30。

---

## 这个产品能做什么

### 像普通团队聊天一样
- 多频道聊天，支持**公开频道**和**私有频道**（私有的只有成员能看）。
- **线程回复**：对某条消息展开讨论，不刷屏。
- **@提及**：输入 @ 自动弹出人/AI 列表（支持中文名）。
- 消息支持 **Markdown 富文本 + 代码高亮**、**发图片/文件**（图片可点开大图，文件可下载）、**编辑已发消息**、**链接自动预览**。
- 消息可加 👍 等表情、可复制、可回复。

### AI Agent 是“真同事”，不只是问答机器人
在「Agent 管理」里可以**创建 / 编辑 / 删除** AI Agent，给它起名字、设角色描述。被 @ 后它会：
- **看懂上下文并回复**（在主频道或线程里）。
- **领任务、改状态**：在任务看板里认领、推进到「审查中/完成」。
- **记得住事情**：每个 Agent 有自己的“记忆”，跨对话记住你的偏好、频道在聊什么、长期事项——不会每次从零开始。
- **会定提醒、主动跟进**：比如让它「2 小时后看看那个事办好没」，到点它会自己回来处理。
- **能传文件**：需要时上传并发出文件/截图。
- **进私有频道**：你在频道里 @ 它，它就自动加入这个频道。

### 用起来顺手
- 浅色 / 深色主题随手切换。
- 网络断了有提示，消息发失败能一键重发，断网时排队、恢复后自动补发。
- 消息很多也不卡（长列表做了优化）。
- 注册时有**密码强度提示**，登录有**防爆破保护**（连续失败会锁一会儿）；可设头像。

## 目前的完成度

✅ **已经能用**（核心功能基本齐全）
- 聊天、频道（公开/私有）、线程、@提及、表情、消息编辑、链接预览
- 发图片/文件、Markdown、富文本
- AI Agent 的创建/编辑/删除、对话、任务、记忆、提醒、传文件、线程内回复
- 主题切换、离线提示与重发、登录安全、个人资料/头像

🚧 **还在路上 / 后续增强**
- AI 回复目前**单次约 30 秒**（在做提速）。
- AI 现在主要是「被 @ 才动」+「定时提醒」，更主动的「自己巡检/发起」还在规划。
- 一些更高级的协作：AI 起草操作让人一键确认、接入外部数据源（如行情/工单系统）、私聊（DM）。
- AI 本身是通用大模型，**没有实时外部数据**（比如实时行情），要查这类信息需要后续给它接专门的工具。

## 怎么快速试一下
1. 在「Agent 管理」建一个 Agent，名字随意、描述写上它的角色（如“前端代码审查”）。
2. 进任意频道，输入 `@它的名字` 加一句话，比如「@小助手 帮我理一下今天的待办」。
3. 等它回复（约 30 秒）。可以接着让它「在本频道建个任务：写周报」「2 小时后提醒我交周报」。
4. 想让它进私有频道，直接在那个私有频道里 @ 它即可。
