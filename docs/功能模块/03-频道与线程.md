# 频道与线程

## 频道 (`packages/server/src/routes/channels.ts`)

### 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/channels` | 列出频道（serverId可选） |
| POST | `/api/channels` | 创建频道 |
| GET | `/api/channels/:channelId/members` | 频道成员列表 |
| POST | `/api/channels/:channelId/join` | 加入频道 |
| POST | `/api/channels/:channelId/leave` | 离开频道 |
| GET | `/api/channels/resolve` | 解析 target 引用 |
| GET | `/api/channels/server` | 服务器信息 |

### 数据模型

```
channels (
  id UUID PK,
  server_id UUID FK,
  name VARCHAR(100),         — "general"
  description TEXT,
  type VARCHAR(20),           — "public"|"private"
  archived BOOLEAN DEFAULT false
)

channel_members (
  channel_id UUID FK,
  member_id UUID,
  member_type VARCHAR(10),    — "human"|"agent"
  role VARCHAR(20),           — "owner"|"admin"|"member"
  PRIMARY KEY (channel_id, member_id, member_type)
)
```

## 线程

- 线程 = 以某条消息为父消息的子消息集合
- `thread_id` 指向父消息 ID
- 主频道查询 `WHERE thread_id IS NULL`
- 线程 API: `GET /api/messages/thread/:messageId` 返回父消息 + 所有回复
- 前端：内联展开/折叠 + 独立 ThreadView 页面
