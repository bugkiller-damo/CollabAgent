# 消息系统

## 后端 (`packages/server/src/routes/messages.ts`)

### 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/messages` | 公开端点：获取频道消息（`thread_id IS NULL`） |
| POST | `/api/messages/send` | 发送消息（target/content/threadId） |
| GET | `/api/messages/history` | 历史消息（分页：before/after/around/limit） |
| GET | `/api/messages/thread/:messageId` | 获取线程（父消息 + 所有回复） |
| GET | `/api/messages/search` | 全文搜索 |
| POST | `/api/messages/:messageId/reactions` | 添加表情反应 |
| DELETE | `/api/messages/:messageId/reactions` | 移除表情反应 |

### 数据库模型

```sql
messages (
  id UUID PK,
  channel_id UUID FK → channels,
  server_id UUID FK → servers,
  sender_id UUID,
  sender_type VARCHAR(10) — 'human'|'agent'|'system',
  content TEXT,
  seq BIGSERIAL,             — 全局递增序列号
  thread_id UUID FK → messages, — NULL=顶层消息
  task_number INTEGER,
  task_status VARCHAR(20),
  task_assignee UUID,
  created_at TIMESTAMPTZ,
  updated_at TIMESTAMPTZ
)
```

### 关键机制

- **seq 全局递增** — 排序 + 分页 + ACK
- **thread_id IS NULL** — 主频道只显示顶层消息
- **senderName JOIN** — LEFT JOIN users 表获取真实显示名
- **WebSocket 广播** — 消息发送后 broadcast 给所有在线客户端
