# CollabAgent 用户体验优化方案

> 聚焦用户端（server + web），不涉及 daemon 开发

---

## 一、新用户上手 (Onboarding)

### 1.1 首次引导
| 优化项 | 说明 | 优先级 |
|---|---|---|
| 注册后引导页 | 首次登录弹出 3 步引导：选头像 → 加入频道 → 发第一条消息 | ⭐⭐⭐ |
| 默认头像生成 | 用户名首字母 + 随机底色，不用上传也能有头像 | ⭐⭐⭐ |
| 示例频道预置 | 除 general/random/engineering 外，加一个 `#welcome` 频道，里面有一条 bot 发的欢迎消息教用户怎么用 | ⭐⭐ |

### 1.2 空状态提示
| 优化项 | 说明 | 优先级 |
|---|---|---|
| 频道列表为空 | 显示 "还没有频道，点击创建" 而不是空白 | ⭐⭐⭐ |
| 消息列表为空 | 显示 "👋 这是 #general 的第一条消息，打个招呼吧" | ⭐⭐⭐ |
| 设置页空字段 | 昵称/简介为空时显示 "点击编辑" 占位符 | ⭐⭐ |

## 二、频道体验 (Channel)

### 2.1 频道管理
| 优化项 | 说明 | 优先级 |
|---|---|---|
| 创建频道 UI | 侧栏 "+" 按钮 → 弹窗输入名称+描述 → 创建 | ⭐⭐⭐ |
| 频道设置 | 点频道名 → 下拉菜单：查看成员 / 修改描述 / 归档 | ⭐⭐ |
| 频道搜索 | 侧栏顶部搜索框，输入过滤频道列表 | ⭐⭐ |
| 频道排序 | 按活跃度 / 字母排序切换 | ⭐ |

### 2.2 频道内体验
| 优化项 | 说明 | 优先级 |
|---|---|---|
| @提及自动补全 | 输入 @ 弹出用户/Agent 列表，上下键选择，回车确认 | ⭐⭐⭐ |
| 消息时间戳 | hover 消息左侧显示精确时间 | ⭐⭐ |
| 消息分组 | 同一用户 5 分钟内连续消息合并头像显示 | ⭐⭐ |
| 未读分隔线 | "以下为新消息" 分割线 | ⭐⭐ |
| 滚动到顶部加载更多 | 上滚自动加载历史消息 | ⭐⭐ |

## 三、消息功能 (Message)

### 3.1 富文本
| 优化项 | 说明 | 优先级 |
|---|---|---|
| Markdown 渲染 | 支持 **粗体** *斜体* `代码` ```代码块``` >引用 -列表 | ⭐⭐⭐ |
| 代码高亮 | ```javascript 自动语言检测和高亮 | ⭐⭐⭐ |

| 表情选择器 | 点击表情按钮弹出 emoji picker | ⭐⭐ |

### 3.2 消息操作
| 优化项 | 说明 | 优先级 |
|---|---|---|
| 消息操作菜单 | hover 消息右侧出现 "⋮" → 复制/回复/转任务/删除 | ⭐⭐⭐ |
| 复制消息 | 一键复制消息文本 | ⭐⭐⭐ |
| 编辑消息 | 自己的消息可编辑（显示 "已编辑" 标记） | ⭐⭐ |
| 删除消息 | 自己的消息可删除 | ⭐⭐ |
| 消息转发 | 转发到其他频道 | ⭐ |

### 3.3 文件与附件
| 优化项 | 说明 | 优先级 |
|---|---|---|
| 拖拽上传 | 拖文件到输入框自动上传 | ⭐⭐⭐ |
| 图片预览 | 发送图片 → 点击放大 → Lightbox 浏览 | ⭐⭐⭐ |
| 粘贴图片 | Ctrl+V 粘贴剪贴板图片直接发送 | ⭐⭐ |
| 文件卡片 | PDF/文档显示文件名+大小+下载按钮 | ⭐⭐ |

## 四、用户系统 (User)

### 4.1 个人资料
| 优化项 | 说明 | 优先级 |
|---|---|---|
| 头像上传 | 设置页上传/裁剪头像 | ⭐⭐⭐ |
| 在线状态 | 绿点 = 在线，灰点 = 离线，黄点 = 离开 | ⭐⭐ |
| 自定义状态 | "在开会" / "专注中" 等状态文字 | ⭐ |
| 个人名片 | hover @用户名 弹出名片卡（头像+昵称+简介+频道） | ⭐⭐ |

### 4.2 通知
| 优化项 | 说明 | 优先级 |
|---|---|---|
| 浏览器通知 | 新消息 → 桌面通知（需授权） | ⭐⭐⭐ |
| @我 高亮 | 频道列表里 @我的消息用红色标记 | ⭐⭐⭐ |
| 通知设置 | 开关：桌面通知 / 声音 / @所有人 | ⭐⭐ |
| 免打扰 | 对特定频道静音 | ⭐⭐ |

## 五、界面美化 (UI/UX)

### 5.1 主题
| 优化项 | 说明 | 优先级 |
|---|---|---|
| 亮色主题 | 除了当前的暗色，加一个亮色切换 | ⭐⭐ |
| 字体大小 | 小/中/大三档 | ⭐ |
| 侧栏折叠 | 点击收起侧栏，只显示图标 | ⭐⭐ |

### 5.2 响应式
| 优化项 | 说明 | 优先级 |
|---|---|---|
| 移动端适配 | 手机浏览器访问自动切换为移动布局 | ⭐⭐⭐ |
| 触屏手势 | 左滑显示侧栏，右滑关闭 | ⭐⭐ |

### 5.3 加载与错误
| 优化项 | 说明 | 优先级 |
|---|---|---|

| 错误提示 Toast | 统一右上角弹窗，不再 console.error | ⭐⭐⭐ |
| 网络断开提示 | 顶部红色横幅 "连接断开，重连中..."（已有）→ 优化视觉 | ⭐⭐ |
| 404 页面 | 美化 "页面不存在" 页面 | ⭐ |

## 六、性能优化

| 优化项 | 说明 | 优先级 |
|---|---|---|
| 消息虚拟列表 | 大量消息时只渲染可视区域 | ⭐⭐⭐ |
| 图片懒加载 | 滚到可见才加载图片 | ⭐⭐ |
| API 请求去重 | 相同请求合并，避免重复调用 | ⭐⭐ |
| 离线缓存 | Service Worker 缓存静态资源，断网也能看到历史消息 | ⭐ |

## 七、搜索与发现

| 优化项 | 说明 | 优先级 |
|---|---|---|
| 全局搜索 | Ctrl+K 打开搜索框，搜消息/频道/用户 | ⭐⭐⭐ |

| 搜索历史 | 记录最近 5 次搜索 | ⭐ |

## 八、安全加固

| 优化项 | 说明 | 优先级 |
|---|---|---|
| 登录限流 | 5次失败锁定15分钟 | ⭐⭐⭐ |
| Token 刷新 | JWT 过期自动无感刷新 | ⭐⭐ |
| XSS 防护 | 消息内容 HTML 转义，禁止脚本注入 | ⭐⭐⭐ |
| 敏感操作确认 | 删除频道/消息时弹窗确认 | ⭐⭐ |

## 九、实施路线图

### Phase 1：核心体验（1-2 周）
```
空状态提示 → @提及补全 → Markdown 渲染 → 消息操作菜单 → 创建频道 UI
```

### Phase 2：互动增强（1-2 周）
```
图片拖拽上传 → 头像上传 → 浏览器通知 → @我高亮 → 代码高亮 → 表情选择器
```

### Phase 3：界面打磨（1 周）
```
骨架屏 → Toast 提示 → 移动端适配 → 全局搜索 → 虚拟列表
```

### Phase 4：安全与性能（1 周）
```
登录限流 → Token 刷新 → XSS 防护 → 离线缓存
```
